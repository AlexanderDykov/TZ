Реализация ТЗ:
Проект выполнен по схеме model-view-controller. Версия Unity 5.4.0f3 (64-bit). 
В качестве view выступает стандартная система UI от Unity-разработчиков.
Роль Model выполняют ScriptableObject, поскольку они удобнее для хранения данных и их изменения, 
которое не повлечет за собой лишних изменений в проекте.
Все контроллеры лежат в папке Assets\Scripts\Controller.
CharacterController - выступает за выгрузку настроек в сцену и их отображение на персонаже.
MenuController - реализовует переход со сцены на сцену с сохранением настроек.
Так же дополнительно было создано SettingsManager - синглтон, предназначеный для переноса настроек с сцены на сцену.
